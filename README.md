**# AI Resume Analyzer**



**This is a \*\*Resume Analysis and Job Matching system\*\* built using Python.**  

**It predicts the category of a resume using machine learning and allows searching for matching skills.**



**## Features**



**- Analyze resumes using a trained Logistic Regression model**

**- Predict categories of resumes**

**- Find resumes that match specific skills**

**- Simple Streamlit web interface**



**## Files**



**- `app.py` → Main Streamlit application**  

**- `model.pkl` → Trained ML model**  

**- `vectorizer.pkl` → TF-IDF vectorizer used for text processing**
